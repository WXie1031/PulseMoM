Examples

adr_ex1.cxx: Demonstrates automatic Jacobian generation using ADOL-C for a nonlinear reaction problem from chemistry.
adr_ex5adj.cxx: Demonstrates adjoint sensitivity analysis for Reaction-Diffusion Equations.
ex16adj.cxx: Demonstrates automatic Jacobian generation using ADOL-C for an adjoint sensitivity analysis of the van der Pol equation.
ex16adj_tl.cxx: Demonstrates tapeless automatic Jacobian generation using ADOL-C for an adjoint sensitivity analysis of the van der Pol equation.
ex16opt_ic.cxx: Demonstrates automatic Jacobian generation using ADOL-C for an ODE-constrained optimization problem.
adr_ex5adj_mf.cxx: Demonstrates automatic, matrix-free Jacobian generation using ADOL-C for a time-dependent PDE in 2d, solved using implicit timestepping.
