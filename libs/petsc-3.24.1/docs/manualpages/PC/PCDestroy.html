PCDestroy#

Destroys PC context that was created with PCCreate().

Synopsis#

#include "petscksp.h" 
PetscErrorCode PCDestroy(PC *pc)

Collective

Input Parameter#

  • pc - the PC preconditioner context

See Also#

KSP: Linear System Solvers, PC, PCCreate(), PCSetUp()

Level#

developer

Location#

src/ksp/pc/interface/precon.c
