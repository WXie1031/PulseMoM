## 目标
- 在保持 MSVC 兼容和现有构建稳定的前提下，补全层状介质 Green 函数的连续谱（同层/跨层）主贡献，并建立可验证的数值校核与性能路径。

## 改动范围
- 仅在 `src/core/electromagnetic_kernels.c/.h` 里实现连续谱近似与内部拆分；不改变对外 API 形态（继续使用 `complex_t`）。
- 在 `apps/mom_cli.c` 增加参数化校核（不依赖求解器），用于快速验证与趋势观察。
- 在 `tests` 目录添加最小单测（双层介质固定参数），在 MSVC 下启用。

## 技术实现
- 连续谱主值近似：
  - 以径向频率 `kρ` 采样的有限区间积分，采用可调节点数与权重（Gauss–Laguerre/简单等距 + 收敛因子）实现稳定近似。
  - 同层与跨层两条路径分别计算，再汇合为 `green_function_layered_media` 的结果。
- 界面与衰减：
  - 保留已实现的传输系数近似 `|T|` 与皮肤深度衰减 `exp(-|z−z'|/δ)`，作为连续谱的幅值调制与稳定项。
- 批处理：
  - 复用已有 `batch_*` 接口，新增内部分批评估函数，避免一次性大数组的内存峰值。

## 接口与兼容
- 无新增对外 API；新增内部静态函数（如 `integrate_continuous_spectrum_same_layer/cross_layer`）。
- 继续使用 `complex_t` 与 C 接口；避免 C 复数与不兼容语法（OpenMP for 初始化遵循 MSVC 标准）。

## 数值校核
- CLI 参数化：
  - 对 `σ、|Z|、厚度、z/z'、ρ、频率` 进行小范围扫描，打印 Re/Im 与相对变化，验证物理趋势与稳定性。
- 单元测试：
  - 固定双层介质与频率的用例，断言 Re/Im 在合理区间（与自由空间相比的衰减与层依赖）。
- 回归基线：
  - 保留自由空间核的有限差分比对作为梯度基线。

## 性能与并行
- 初期串行实现；必要时对 `kρ` 积分采用 OpenMP 并行（标准 for 初始化），并保留禁用开关。
- 为频繁评估引入插值表缓存（`kρ` 采样点与权重），降低重复计算开销。

## 风险与回退
- 若连续谱在某些参数下不稳定：回退到“自由空间 + 传输系数 + 衰减”的近似，接口不变；记录参数边界并在 CLI 提示。
- 所有新增逻辑提供裁剪与下限（如 `exp(-10)`）以防数值下溢/上溢。

## 里程碑
- 里程碑 1：同层主值近似与 CLI 校核通过（构建 0 错误）。
- 里程碑 2：跨层主值近似与插值表缓存；批处理评估通过。
- 里程碑 3：参数化测试覆盖与性能小结；准备逐步恢复其它核心模块。
