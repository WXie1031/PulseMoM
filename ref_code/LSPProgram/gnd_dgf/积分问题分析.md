# 积分问题分析报告

## 对比dgf-strata-main的实现

根据dgf-strata-main的代码（`sommerfeld_integrals.hpp`第135行），Sommerfeld积分的被积函数是：

```cpp
return Jv*K*std::pow(krho, order+1)/2.0/M_PI;
```

其中：
- **J0 (order=0)**: `krho^1 = krho`
- **J1 (order=1)**: `krho^2`
- 最后除以 `2π`

## 当前MATLAB代码分析

### 1. Gxx (使用J0) ✓ 正确
```matlab
Vih_J0_tmp = (Vih_tmp_sub-Vih_Asub_tmp_sub).*J0_tmp.*repmat(kp_int,Nxx,1)*dkp;
```
- 有 `kp_int`，所以是 `J0 * kp * dkp` ✓

### 2. Gzx (使用J1) ✗ **错误！**
```matlab
Iih_J1_tmp = (1/2*(-Ah1+Ah2+Ah3-Ah4)-Iih_Bsub_tmp_sub).*J1_tmp*dkp;
Iie_J1_tmp = (1/2*(-Ae1+Ae2+Ae3-Ae4)-Iie_Bsub_tmp_sub).*J1_tmp*dkp;
```
- **缺少 `kp_int`！**
- 应该是 `J1 * kp^2 * dkp`，但代码中只有 `J1 * dkp`
- 这导致Gzx的结果不正确

### 3. Gzz (使用J0) - 需要验证
- `Ivh_J0_tmp`: `J0/kp * dkp`
- `Ive_J0_tmp_A`: `J0 * kp * dkp`
- `Ive_J0_tmp_B`: `J0/kp * dkp`
- 这些项的组合需要根据具体公式验证

### 4. Gphi (使用J0) - 需要验证
- `Vie_J0_kp_tmp`: `J0/kp * dkp`
- `Vih_J0_kp_tmp`: `J0/kp * dkp`
- 需要根据具体公式验证

## 问题根源

Gzx的积分核缺少 `kp^2` 因子，这会导致：
1. 积分结果量纲不正确
2. 数值结果错误
3. 与参考实现不一致

## 修复方案

在Gzx的积分核中添加 `kp_int^2` 因子。

### 已修复的文件

1. `dgf_main_sub_same_sub_i.m` - 同层中间层
2. `dgf_main_sub_same_sub_n.m` - 同层最后一层
3. `dgf_main_sub_same_sub_1.m` - 同层第一层
4. `dgf_main_num_same.m` - 数值方法同层
5. `dgf_main_num_diff_fro_i.m` - 不同层前向
6. `dgf_main_num_diff_bak_i.m` - 不同层后向

### 修复内容

**修复前**:
```matlab
Iih_J1_tmp = (...).*J1_tmp*dkp;
Iie_J1_tmp = (...).*J1_tmp*dkp;
```

**修复后**:
```matlab
Iih_J1_tmp = (...).*J1_tmp.*repmat(kp_int.^2,Nf,1)*dkp;
Iie_J1_tmp = (...).*J1_tmp.*repmat(kp_int.^2,Nf,1)*dkp;
```
