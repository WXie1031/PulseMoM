# 分层格林函数代码修复总结

## 修复的文件列表

1. `dgf_main.m` - 主函数，修复了多个关键问题
2. `dgf_main_sub_same_sub_i.m` - 同层计算函数，修复了维度不匹配和计算错误

## 详细修改内容

### 1. dgf_main.m

#### 修改1: 修复 epr_lyr 变量覆盖问题
**位置**: 第45-46行
**修改前**:
```matlab
epr_lyr = epr_lyr - 1j.*sig_lyr./(w0*ep0);
ep_lyr = ep0*epr_lyr;
```
**修改后**:
```matlab
% Convert relative permittivity to complex permittivity (including conductivity)
epr_lyr_complex = epr_lyr - 1j.*sig_lyr./(w0*ep0);
ep_lyr = ep0*epr_lyr_complex;
```

#### 修改2: 移除硬编码的 dkp
**位置**: 第65行
**修改前**:
```matlab
dkp=20;
```
**修改后**:
```matlab
% dkp=20;  % Commented out: hard-coded value overrides calculated dkp
```

#### 修改3: 修正相邻层 dz 计算
**位置**: 第214-238行
**修改前**:
```matlab
zf_tmp = repmat(zbdy(id_s),Ntmp,1);
```
**修改后**:
```matlab
% Use actual field point z coordinates, not boundary values
zf_adj_fro = zf(idf_adj_fro);
zf_adj_bak = zf(idf_adj_bak);
```

#### 修改4: 确保逻辑互斥
**位置**: 第241, 253行
**修改前**:
```matlab
idf_oth_fro = (id_f<id_s);
idf_oth_bak = (id_f>id_s);
```
**修改后**:
```matlab
idf_oth_fro = (id_f<id_s) & ~idf_adj_fro;
idf_oth_bak = (id_f>id_s) & ~idf_adj_bak;
```

#### 修改5: 添加相邻层处理代码
**位置**: 第265-296行
**修改前**: 相邻层情况未被处理
**修改后**: 添加了完整的相邻层处理逻辑
```matlab
% Process adjacent front layer case (field point in layer id_s-1)
if any(idf_adj_fro)
    [Gxx(ik,idf_adj_fro),Gzx(ik,idf_adj_fro),Gzz(ik,idf_adj_fro),Gphi(ik,idf_adj_fro)] = ...
        dgf_main_sub_diff_fro(...);
end

% Process adjacent back layer case (field point in layer id_s+1)
if any(idf_adj_bak)
    [Gxx(ik,idf_adj_bak),Gzx(ik,idf_adj_bak),Gzz(ik,idf_adj_bak),Gphi(ik,idf_adj_bak)] = ...
        dgf_main_sub_diff_bak(...);
end
```

#### 修改6: 优化逻辑索引使用
**位置**: 第262, 279, 289行
**修改前**:
```matlab
if find(idf_same==1)
```
**修改后**:
```matlab
if any(idf_same)
```

#### 修改7: 更新函数调用参数
**位置**: 多处
**修改**: 将所有 `epr_lyr` 参数改为 `epr_lyr_complex`

### 2. dgf_main_sub_same_sub_i.m

#### 修改1: 修复维度不匹配
**位置**: 第150行
**修改前**:
```matlab
[Bsub_e0,Bsub_e1,Bsub_e2,Bsub_e3,Bsub_e4] = dgf_kernal_Bsub(dz0,dz1,dz2,dz3,dz4,...
    Re_ij_inf(id_s),Re_ji_inf(id_s-1), kp_int);
```
**修改后**:
```matlab
[Bsub_e0,Bsub_e1,Bsub_e2,Bsub_e3,Bsub_e4] = dgf_kernal_Bsub(dz0(indzx),dz1(indzx),dz2(indzx),dz3(indzx),dz4(indzx),...
    Re_ij_inf(id_s),Re_ji_inf(id_s-1), kp_int);
```

#### 修改2: 修复 Gzz_tail 计算错误
**位置**: 第333行
**修改前**:
```matlab
+ mur_lyr(id_s)*epr_lyr(id_s)./(2*epr_lyr(id_s)) .* (Ct_e0-Ct_e1-Ct_e2+Ct_e3+Ct_e4);
```
**修改后**:
```matlab
+ mur_lyr(id_s)/2 .* (Ct_e0-Ct_e1-Ct_e2+Ct_e3+Ct_e4);  % Fixed: epr_lyr(id_s)./(2*epr_lyr(id_s)) = 1/2
```
