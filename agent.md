## RWG 表面电流改造总览（Agent Map）

- **目标**: 将现有 MoM 求解从“单元脉冲基 + 单元常数电流”升级为“RWG 边基 + 连续表面电流评估”，并在 VTK 层面输出物理上更连续、可用于展示的表面电流分布。
- **当前状态**: 
  - Unified solver 内部已经有 RWG 相关的数据结构和积分实现片段，但 CLI 路径使用的是“每三角形一个未知”的简化版本。
  - VTK 导出目前通过单元常数 + 顶点平均 + 轻微平滑来构造 POINT_DATA，平滑度仍有限。

### 分解任务

1. **RWG 基函数设计与接口调整**（见 `agent_rwg_basis.md`）
2. **矩阵装配与求解流程切换到 RWG**（见 `agent_rwg_assembly.md`）
3. **表面电流评估与 VTK 导出（基于 RWG 展开）**（见 `agent_rwg_postproc.md`）
4. **回归测试与对比验证（脉冲基 vs RWG）**（预留，待前三项完成后补充）

每个子任务文件中包含：目标、关键修改点（文件/函数）、实施步骤和检查项。
