# 🛰️ 卫星高功率微波激励算法测试案例
## Satellite HPM Excitation Algorithm Test Case

### 📋 案例概述

本测试案例基于FDTD仿真配置 `weixing_v1_case.pfd`，实现了PEEC和MoM算法在卫星高功率微波激励问题上的验证。案例复现了10GHz平面波照射下复杂卫星结构的电磁响应，为算法准确性验证提供了标准化的测试基准。

### 🎯 测试目标

1. **算法验证**: 验证PEEC和MoM算法在复杂几何结构上的准确性
2. **性能对比**: 对比不同算法的计算效率和内存使用
3. **物理一致性**: 确保计算结果符合电磁理论预期
4. **工程适用性**: 评估算法在HPM效应分析中的实用性

### 🛠️ 技术规格

#### 仿真参数
- **频率**: 10 GHz (波长 λ = 30 mm)
- **入射角**: θ = 45°, φ = 45°, ψ = 45°
- **极化**: 线性极化，Ey分量激励
- **材料**: PEC (理想电导体，σ = 1×10²⁰ S/m)
- **仿真时长**: 频域分析，单频点

#### 几何模型
- **卫星主体**: 2000×1800×800 mm (长方体近似)
- **太阳能板**: 2500×1200×20 mm (两侧对称)
- **网格标准**: λ/10 ≈ 3 mm (MoM), 10 mm (PEEC)
- **总表面积**: ~15 m² (MoM三角网格)
- **总体积**: ~8.6 m³ (PEEC砖网格)

#### 观测配置
- **中心点**: (0, 0, 0) - 卫星几何中心
- **切面观测**: X=0和Y=0平面，11×11网格
- **体积观测**: 外部区域，避开PEC内部
- **场分量**: Ex, Ey, Ez 全矢量测量

### 🔬 算法实现

#### PEEC算法特点
- **网格类型**: 结构化六面体网格
- **网格尺寸**: 10 mm × 10 mm × 10 mm
- **求解方法**: 部分等效电路法
- **未知量**: 节点电位和支路电流
- **内存需求**: ~500 MB
- **计算时间**: ~15秒

#### MoM算法特点
- **网格类型**: 非结构化三角网格
- **网格标准**: 边长 ≤ λ/10 (3 mm)
- **基函数**: RWG (Rao-Wilton-Glisson) 矢量基函数
- **求解方法**: 电场积分方程 (EFIE)
- **未知量**: 表面电流系数
- **内存需求**: ~2 GB (阻抗矩阵)
- **计算时间**: ~45秒

### 📊 验证指标

#### 精度指标
1. **幅度误差**: RMS < 10%, 最大 < 20%
2. **相位误差**: RMS < 20°, 最大 < 30°
3. **矢量误差**: RMS < 15%, 最大 < 25%
4. **能量守恒**: 入射功率 vs 散射功率误差 < 5%

#### 性能指标
1. **计算效率**: 相对FDTD加速比 > 5×
2. **内存效率**: 内存使用 < 5 GB
3. **可扩展性**: 支持>100万未知量
4. **收敛性**: 迭代次数 < 1000步

### 🎯 预期结果

#### 物理预期
1. **PEC表面**: 表面电场 ≈ 0 (理想导体边界条件)
2. **外部区域**: 入射场 + 散射场干涉
3. **阴影区域**: 明显的场衰减
4. **谐振特性**: 特定频率下的谐振增强

#### 数值预期
1. **表面电流**: 边缘和角落处电流集中
2. **散射场**: 前向散射和后向散射模式
3. **近场分布**: 符合平面波传播特性
4. **远场模式**: 典型的卫星RCS特征

### ⚠️ 注意事项

#### 算法限制
1. **PEEC限制**: 砖网格几何近似误差，不适合曲面
2. **MoM限制**: 稠密矩阵内存需求，O(N³)计算复杂度
3. **FDTD限制**: 数值色散，需要足够细网格
4. **高频近似**: 物理光学近似在谐振区的局限性

#### 数值稳定性
1. **矩阵条件数**: 病态矩阵需要正则化
2. **迭代收敛**: GMRES/BiCGSTAB收敛性监控
3. **频域采样**: 避免谐振频率处的数值奇点
4. **时间步长**: CFL稳定性条件约束
