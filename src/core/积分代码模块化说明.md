# 积分代码模块化说明

## 概述

为了更好的代码组织和维护性，积分代码已经按功能模块化组织。当前实现采用渐进式模块化策略，将工具函数提取到独立模块，主文件保留核心实现。

## 模块结构

### 1. integration_utils.c / integration_utils.h
**功能**: 数值积分工具函数

**包含函数**:
- `gauss_quadrature_1d()` - 1D高斯积分点和权重
- `compute_distance_3d()` - 3D距离计算
- `vector3d_dot()` - 向量点积
- `vector3d_cross()` - 向量叉积
- `get_triangle_centroid()` - 三角形重心
- `get_opposite_vertex()` - 获取对边顶点
- `make_complex()` - 复数创建辅助函数

**用途**: 被所有其他积分模块使用的基础工具函数

### 2. electromagnetic_kernels.c / electromagnetic_kernels.h
**功能**: 电磁核函数主实现文件

**包含模块**:
- **Green's Functions** (行 75-140)
  - `green_function_free_space()` - 自由空间Green函数
  - `green_function_gradient_free_space()` - Green函数梯度
  - `green_function_layered_media()` - 多层介质Green函数
  - `green_function_layered_dcim()` - DCIM加速
  - `green_function_layered_fast()` - 快速评估
  - `green_function_periodic()` - 周期结构Green函数

- **Single Integrals** (行 183-330)
  - `integrate_triangle_singular()` - 三角形奇异积分
  - `integrate_rectangle_regular()` - 矩形规则积分
  - `integrate_wire_thin()` - 细线积分

- **Singularity Treatment** (行 332-405)
  - `singularity_extraction_1overr()` - 1/r奇异性提取
  - `edge_singularity_treatment()` - 边奇异性处理（Duffy变换）

- **PEEC Double Integrals** (行 467-786)
  - `integrate_surface_surface_capacitance()` - 表面-表面电容积分
  - `integrate_surface_surface_inductance()` - 表面-表面电感积分
  - `integrate_wire_wire_neumann()` - 线-线Neumann公式
  - `integrate_quad_quad_double()` - 四边形-四边形双重积分
  - `integrate_point_surface()` - 点-表面积分
  - `integrate_point_wire()` - 点-线积分
  - `integrate_wire_surface()` - 线-表面积分

- **MoM RWG Integrals** (行 788-1400+)
  - `integrate_rwg_rwg_efie()` - RWG基函数EFIE积分
  - `integrate_rwg_rwg_mfie()` - RWG基函数MFIE积分
  - `integrate_rwg_rwg_cfie()` - RWG基函数CFIE积分
  - `integrate_rwg_plane_wave()` - RWG平面波激励积分
  - `integrate_rwg_point_source()` - RWG点源激励积分

## 未来模块化计划

如果代码继续增长，可以考虑进一步拆分为：

1. **greens_functions.c** - 所有Green函数实现
2. **single_integrals.c** - 单重积分（三角形、矩形、线）
3. **singularity_treatment.c** - 奇异性处理
4. **double_integrals_peec.c** - PEEC双重积分
5. **rwg_integrals_mom.c** - MoM RWG积分

## 编译配置

所有模块文件已在 `CMakeLists.txt` 中配置：
- `integration_utils.c` - 添加到 CORE_SOURCES
- `electromagnetic_kernels.c` - 主实现文件

## 使用说明

1. **工具函数**: 通过 `#include "integration_utils.h"` 使用
2. **积分函数**: 通过 `#include "electromagnetic_kernels.h"` 使用
3. **依赖关系**: `electromagnetic_kernels.c` 依赖 `integration_utils.h`

## 命名规范

- **文件命名**: 使用小写字母和下划线，如 `integration_utils.c`
- **函数命名**: 使用功能前缀，如 `integrate_*`, `green_function_*`, `compute_*`
- **模块划分**: 按功能领域划分，避免按技术细节划分

## 维护建议

1. 新增工具函数应添加到 `integration_utils.c`
2. 新增积分函数应根据功能添加到相应模块
3. 如果单个模块超过1000行，考虑进一步拆分
4. 保持模块间的低耦合，高内聚
