### 1. 商用软件max_facets设置分析

在商用电磁仿真软件（如CST、HFSS、FEKO）中，**max_facets（最大面片数）通常没有硬性限制**，而是采用以下专业设置策略：

#### 专业网格划分标准：
- **标准精度**：λ/10（10GHz时约3mm边长）
- **高精度**：λ/20（10GHz时约1.5mm边长）  
- **极高精度**：λ/50（10GHz时约0.6mm边长）

#### 本实现采用的方法：
```python
# 标准EM网格划分：λ/10, 高精度: λ/20, 极高精度: λ/50
if mesh_accuracy == 'standard':
    target_edge_length = self.wavelength / 10
elif mesh_accuracy == 'high':
    target_edge_length = self.wavelength / 20

    target_edge_length = self.wavelength / 50
```

**结论**：商用软件根据波长自动确定网格密度，而不是设置固定的最大面片数。

### 2. MoM和PEEC时域计算方法

#### MoM时域计算（TD-MoM）：
- **方法**：逆傅里叶变换（IFFT）
- **原理**：频域阻抗矩阵求解 → 频域响应 → IFFT → 时域响应
- **实现**：
```python
def calculate_time_domain_mom(self, rwg_functions, frequency_response, time_signal):
    """时域MoM计算 - 使用逆傅里叶变换"""
    # 频率响应到脉冲响应
    full_freq_response = np.concatenate([frequency_response, np.conj(frequency_response[1:][::-1])])
    # 逆傅里叶变换到时域
    impulse_response = ifft(full_freq_response)
    return time_response[:len(time_signal)]
```

#### PEEC时域计算（TD-PEEC）：
- **方法**：时域积分方程（MOT - Marching-On-in-Time）
- **原理**：等效电路时域求解，R-L-P参数直接时域推进
- **实现**：基于电路方程的时步推进算法

### 3. 散射场计算目的与物理意义

#### 问题背景：
平面波入射到卫星，卫星表面产生表面电流，卫星内部有场辐射进去。

#### 散射场的物理意义：
- **总场分解**：**E_total = E_incident + E_scattered**
- **散射场**：由物体表面电流产生的二次辐射场
- **计算目的**：
  1. **雷达散射截面（RCS）分析**
  2. **电磁耦合评估**
  3. **屏蔽效能计算**
  4. **天线方向图影响**

#### 对应关系：
- **入射场**：平面波直接场（已知）
- **散射场**：卫星表面电流产生的辐射场（计算得到）
- **总场**：实际测量的复合场
